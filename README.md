    <b>Description</b><br/>
    <p>A collection of easy-to-digest code examples for Lightning Web Components. From Hello World to data access and third-party libraries, there is a recipe for that!</p>
    <b>Features</b><br/>
    <p>
      <code>Lightning Web Components</code>, <code>Lightning Data Service</code>, <code>Lightning Message Service</code>, <code>Navigation</code>, <code>@wire</code>, <code>Composition</code>, <code>Events</code>, <code>3rd Party Library</code>
